## How to Run

1. Clone the repository

```bash
git clone https://github.com/Rikeshhsinha/C-LANGUAGE.git
```

2. Open the project in VS Code

3. Compile the program

```bash
gcc filename.c -o output
```

4. Run the program

```bash
./output
```
